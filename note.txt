Git and GitHub
-------------

version control system

remote repository - github storage
local repository  - system storage

Git operations
--------------

push  - local repo to remote repo  - upload new project and updates
clone - remote repo to local repo  - downloading exact copy
pull  - remote to local            - downloading new updates


push
